# 大理旅游线路规划平台

> 一个智能化的旅游线路规划平台，以大理为首个试点城市，提供AI生成行程、地图可视化、社区分享等功能。

## 📋 项目简介

随着自由行旅游的普及，游客在规划行程时面临以下痛点：
- 景点信息分散，需要在多个平台间切换查找
- 行程安排缺乏合理性，容易出现路线绕远、时间冲突
- 难以获取真实的本地游玩经验和推荐

本项目旨在打造一个智能化的旅游线路规划平台，提供一站式的行程规划服务。

## ✨ 核心功能

- **AI行程生成**：根据天数、主题偏好自动生成合理的多日游行程
- **地图可视化**：在地图上展示POI点和路线，包含步行/驾车/公交耗时
- **社区分享**：用户可以分享行程到社区，形成UGC内容生态
- **后台管理**：管理员可以维护POI数据和社区内容
