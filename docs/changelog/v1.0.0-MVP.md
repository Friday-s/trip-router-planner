# v1.0.0-MVP 版本发布记录

> 发布日期：2026-01-28
> 版本类型：MVP (最小可行产品)

---

## 版本概述

本版本为旅游线路规划平台的首个MVP版本，实现了核心功能框架，以云南大理为示范城市，验证了基于AI大模型的智能行程推荐方案的可行性。

## 功能清单

## 数据库结构

本版本包含9张核心数据表：

| 表名 | 说明 | 记录数 |
|------|------|--------|
| users | 用户表 | 1 (管理员) |
| pois | 景点表 | 16 |
| itineraries | 行程主表 | - |
| itinerary_days | 行程天表 | - |
| itinerary_items | 行程项表 | - |
| route_cache | 路线缓存表 | - |
| posts | 帖子表 | - |
| likes | 点赞表 | - |
| favorites | 收藏表 | - |

## 配置说明

### 环境变量
| 变量名 | 说明 | 默认值 |
|--------|------|--------|
| DB_USERNAME | 数据库用户名 | root |
| DB_PASSWORD | 数据库密码 | (空) |
| REDIS_HOST | Redis地址 | localhost |
| REDIS_PORT | Redis端口 | 6379 |
| DEEPSEEK_API_KEY | DeepSeek API密钥 | 需配置 |
| AMAP_KEY | 高德地图Key | 需配置 |

## 已知问题
