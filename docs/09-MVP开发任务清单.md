## 四、Phase 2：后端核心服务

### 4.1 项目初始化

- [ ] **T2-1** 创建 Spring Boot 项目

### 4.2 认证模块

- [ ] **T2-5** 实现用户注册接口 `POST /api/auth/register`
- [ ] **T2-6** 实现用户登录接口 `POST /api/auth/login`
- [ ] **T2-7** 实现获取当前用户接口 `GET /api/auth/me`
- [ ] **T2-8** 实现 JWT 拦截器

### 4.3 POI 模块

- [ ] **T2-9** 实现 POI 列表接口 `GET /api/pois`
- [ ] **T2-10** 实现 POI 详情接口 `GET /api/pois/{id}`
- [ ] **T2-11** 实现区域列表接口 `GET /api/pois/areas`
- [ ] **T2-12** 实现标签列表接口 `GET /api/pois/tags`

### 4.4 行程模块（核心）

- [ ] **T2-13** 实现推荐引擎接口层（RecommendationEngine）
- [ ] **T2-14** 实现 AI 推荐引擎（AIRecommendationEngine）
- [ ] **T2-15** 实现高德路线服务（AmapService）
- [ ] **T2-16** 实现行程生成接口 `POST /api/itineraries/generate`
- [ ] **T2-17** 实现行程详情接口 `GET /api/itineraries/{id}`
- [ ] **T2-18** 实现保存行程接口 `POST /api/itineraries/{id}/save`
- [ ] **T2-19** 实现我的行程列表接口 `GET /api/itineraries/my`
- [ ] **T2-20** 实现删除行程接口 `DELETE /api/itineraries/{id}`

### 4.5 验收标准

- 所有接口可通过 Postman/curl 正常调用
- 行程生成接口返回完整的行程数据（含每日主题、POI、路线）
- JWT 认证正常工作
